## 📗 Ручная установка: Шаг 1. Установка операционной системы

---

В данном руководстве рассматривается установка Ubuntu Server 24.04 LTS. Это фундамент, на который мы развернем панель WebServerCP.

> ⚠️ Подготовка: Скачайте **[образ Ubuntu Server 24.04 LTS](https://releases.ubuntu.com/24.04.4/ubuntu-24.04.4-live-server-amd64.iso)** и создайте загрузочную флешку (например, с помощью Rufus).

---

### 1. Загрузка и запуск

После загрузки с USB-носителя откроется меню GRUB.

**Действие:** Выберите `Try or Install Ubuntu Server` и нажмите `Enter`.

![GRUB Boot Menu](./images/01.png)

---

### 2. Выбор языка интерфейса

Выберите язык, на котором будет проходить процесс установки.

**Действие:** Выберите `English` (рекомендуется для серверных систем) и подтвердите выбор.

![Language Selection](./images/02.png)

---

### 3. Настройка раскладки клавиатуры

**Действие:** Убедитесь, что выбрана раскладка English (US). Нажмите `[ Done ]`.

![Keyboard Configuration](./images/03.png)

---

### 4. Тип установки

**Действие:** Выберите стандартный `Ubuntu Server`. Нажмите `[ Done ]`.

![Installation Type](./images/04.png)

---

### 5. Настройка сети

Серверу необходим стабильный доступ в интернет для загрузки пакетов.

**Действие:**
- Выберите ваш сетевой интерфейс -> Edit IPv4.
- Рекомендуется выбрать метод `Manual` и указать статический IP, чтобы адрес панели не менялся.
- Укажите вашу подсеть (Subnet), IP адрес, шлюз (Gateway) и DNS (например, 8.8.8.8).
- Нажмите `[ Save ]`, а затем `[ Done ]`.

![Network Configuration](./images/05.png)

---

### 6. Прокси и Зеркала (Proxy & Mirror)

**Действие:**

- Если прокси не требуется, оставьте поле пустым и нажмите `[ Done ]`.

![Proxy Configuration](./images/06.png)

- На этапе выбора зеркала дождитесь статуса passed (проверка связи с репозиторием) и нажмите `[ Done ]`.

![Ubuntu Archive Mirror Configuration](./images/07.png)

---

### 7. Разметка диска (Storage)

Для простоты управления мы будем использовать стандартную разметку.

**Действие:**
- Выберите `Use an entire disk`.
- Важно: Снимите галочку с `Set up this disk as an LVM group`. Это упростит работу с диском в будущем.

![Guided Storage Configuration](./images/08.png)

- Нажмите `[ Done ]`, проверьте итоговую таблицу разделов и подтвердите форматирование в красном окне (выберите `Continue`).

![Storage Configuration Summary](./images/09.png)

---

### 8. Создание профиля пользователя

**Действие:**
- Заполните ваше имя, имя сервера (Hostname) и логин.
- Укажите надежный пароль (он понадобится для команд sudo).
- Нажмите `[ Done ]`.

![Profile Configuration](./images/10.png)

---

### 9. Выбор Ubuntu Pro (опционально)

Ubuntu Pro предлагает расширенную поддержку обновлений, но для работы панели она не обязательна.

**Действие:**
- Выберите `Skip for now`, если не планируете использовать подписку сейчас.
- Нажмите `[ Continue ]`.

**Примечание:** Активировать сервис можно в любой момент позже командой pro attach.

![Upgrade to Ubuntu Pro](./images/11.png)

---

### 10. Настройка SSH-доступа

Это критический шаг для удаленного управления сервером.

**Действие:**
- Отметьте `[X] Install OpenSSH server`.
- Нажмите `[ Done ]`.

![SSH Configuration](./images/12.png)

---

### 11. Дополнительные пакеты (Snaps)

**Действие:**
- Ничего не выбирайте из списка предложенных сервисов.
- Примечание: Для корректной работы `WebServerCP` нам нужна «чистая» система без лишних контейнеров.
- Нажмите `[ Done ]`.

![Featured Server Snaps](./images/13.png)

---

### 12. Завершение установки

Дождитесь окончания процесса (появится надпись `Install complete!`).

**Действие:**
- Нажмите [ Reboot Now ].

![Installation Complete](./images/14.png)

- Когда появится надпись `Please remove the installation medium`, извлеките флешку и нажмите `Enter`.
- Примечание: Ошибку `Failed unmounting cdrom` можно игнорировать.

![Installation Complete](./images/15.png)
